Test

Le solde(balance) a-t-il un effet sur le dpt(deposit) terme?
H0: le solde na aucun effet sur le dpt terme
H1: le solde a un effet sur le dpt terme 

with(data=bankData,t.test(balance[deposit=="no"],balance[deposit=="yes"],var.equal=TRUE))
## 
##  Two Sample t-test
## 
## data:  balance[deposit == "no"] and balance[deposit == "yes"]
## t = -8.5988, df = 11160, p-value < 2.2e-16
## alternative hypothesis: true difference in means is not equal to 0
## 95 percent confidence interval:
##  -643.5002 -404.5814
## sample estimates:
## mean of x mean of y 
##  1280.227  1804.268

La p-valeur est largement infrieure 5%. Donc, on rejette lhypothse nulle.

Prdiction

Le modle de prediction utilis cest une arbre de decision, on veut predire si un client va souscrire ou non un dpt terme( deposit), en prenant 16 variables du jeu de donnees(sauf la variable deposit, cest elle quand veut predire), pour les variables de type qualitatif on va les encoder afin de les utiliser dans larbre de decision. la librarie library(rpart.plot) permet de genrer la rgle de decision pour les variables intrssant.
